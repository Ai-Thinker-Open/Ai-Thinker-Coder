# WS2812 RGB LED 控制 API 参考

> 来源文件：`components/stage/ws2812/ws2812.h`（核心）  
> `ws281x_spi.h`（SPI 模式）/`ws281x_ir.h`（IR 模式）  
> 支持 SPI 和 IR 两种硬件驱动方式的 WS2812/NeoPixel 灯带控制库。

---

## 概述

WS2812 是单线总线 RGB LED，BL602 通过两种方式驱动：
- **SPI 模式**：高速 SPI 输出时序，对 CPU 负载低
- **IR 模式**：利用 IR 红外模块的 PWM 输出时序（管脚固定为 GPIO11）

BL602 的 `ws2812` 库基于颜色模式抽象层，支持 HSV 颜色空间全局亮度控制。

---

## 头文件

```c
#include "ws2812.h"

// SPI 模式（需定义 WS281X_SPI_MODE）
#include "ws281x_spi.h"

// IR 模式（需定义 WS281X_IR_MODE）
#include "ws281x_ir.h"
```

## 类型定义

### `color_t`

颜色结构体（来源 `color_mode.h`）：

```c
typedef struct {
    uint8_t r;   // 红色分量 (0-255)
    uint8_t g;   // 绿色分量 (0-255)
    uint8_t b;   // 蓝色分量 (0-255)
} color_t;
```

### `ws2812_dev_t`

单个 LED 设备描述符：

```c
typedef struct {
    color_t color;      // 当前颜色
    float *brightness;  // 指向全局亮度（指针实现共享）
    uint8_t index;      // LED 索引
} ws2812_dev_t;
```

### `ws2812_strip_t`

整条灯带描述符：

```c
typedef struct {
    ws2812_dev_t *dev;    // LED 数组
    uint8_t led_count;    // LED 数量
    float brightness;     // 全局亮度 (0.0-1.0)
    uint8_t pin;          // GPIO 管脚（SPI 模式使用）
} ws2812_strip_t;
```

## 全局变量

```c
extern ws2812_strip_t *ws2812_strip_dev;
```

> 初始化后指向内部分配的灯带结构体。

## 核心函数

### `ws2812_init`

初始化灯带（自动选择 SPI 或 IR 模式）。

```c
void ws2812_init(ws2812_strip_t *ws2812_strip);
```

| 参数 | 说明 |
|------|------|
| `ws2812_strip` | 灯带配置结构体指针，`led_count` 和 `pin` 必须设置 |

> 内部会根据编译宏 `WS281X_SPI_MODE` 或 `WS281X_IR_MODE` 选择驱动。

---

### `ws2812_set_all_pixels_color`

设置所有 LED 为同一颜色。

```c
void ws2812_set_all_pixels_color(uint8_t r, uint8_t g, uint8_t b, float brightness);
```

| 参数 | 说明 |
|------|------|
| `r/g/b` | RGB 分量 (0-255) |
| `brightness` | 全局亮度 (0.0-1.0)，设为 1.0 则不受 `brightness` 影响 |

---

### `ws2812_show_leds`

将颜色数据刷新到灯带（发送 DMA/IR 帧）。

```c
void ws2812_show_leds(void);
```

> 所有 `set_pixel_color` 调用后必须调用此函数才会实际显示。

---

### `ws2812_set_pixel_color`

设置单个 LED 的颜色（内存中，不自动刷新）。

```c
void ws2812_set_pixel_color(uint8_t index, uint8_t r, uint8_t g, uint8_t b);
```

| 参数 | 说明 |
|------|------|
| `index` | LED 位置（从 0 开始） |
| `r/g/b` | RGB 分量 (0-255) |

---

### `ws2812_set_pixel_brightness`

设置单个 LED 的亮度系数（覆盖全局亮度）。

```c
void ws2812_set_pixel_brightness(uint8_t index, float brightness);
```

| 参数 | 说明 |
|------|------|
| `index` | LED 位置 |
| `brightness` | 亮度系数 (0.0-1.0) |

---

### `ws2812_set_global_brightness`

设置所有 LED 的全局亮度（修改后需重新 `show`）。

```c
void ws2812_set_global_brightness(float brightness);
```

| 参数 | 说明 |
|------|------|
| `brightness` | 全局亮度 (0.0-1.0) |

---

### `ws2812_set_pixel_color_hsv`

设置单个 LED 颜色（HSV 色彩空间，自动应用亮度）。

```c
void ws2812_set_pixel_color_hsv(uint8_t index, uint8_t h, uint8_t s, uint8_t v);
```

| 参数 | 说明 |
|------|------|
| `index` | LED 位置 |
| `h` | 色相 (0-255，对应 0-360°) |
| `s` | 饱和度 (0-255) |
| `v` | 明度 (0-255) |

---

### `ws2812_set_led_count`

运行时修改 LED 数量（会重新分配内存）。

```c
void ws2812_set_led_count(uint8_t count);
```

---

### `ws2812_get_led_count`

获取当前 LED 数量。

```c
uint8_t ws2812_get_led_count(void);
```

## 使用示例

### SPI 模式驱动 8 颗 WS2812

```c
#include "ws2812.h"
#include "ws281x_spi.h"

// 定义灯带配置
static ws2812_strip_t g_ws2812_strip = {
    .led_count = 8,
    .pin = 10,  // GPIO10 作为 SPI MOSI
    .brightness = 0.5,
};

void ws2812_spi_demo(void)
{
    // 使用 SPI 模式
#ifdef WS281X_SPI_MODE
    ws281x_spi_init(&g_ws2812_strip);

    // 设置第 0 颗为红色
    ws281x_spi_set_pixel_color(0, 255, 0, 0);

    // 设置所有 LED 为绿色
    ws2812_set_all_pixels_color(0, 255, 0, 1.0);

    // 刷新显示
    ws281x_spi_show_leds();
#endif
}
```

### IR 模式彩虹呼吸灯

```c
#include "ws2812.h"
#include "ws281x_ir.h"

static ws2812_strip_t g_strip = {
    .led_count = 8,
    .brightness = 0.8,
};

void rainbow_breathing(void)
{
#ifdef WS281X_IR_MODE
    ws281x_ir_init(&g_strip);

    while (1) {
        // HSV 彩虹效果
        for (uint8_t h = 0; h < 255; h += 5) {
            for (uint8_t i = 0; i < g_strip.led_count; i++) {
                ws2812_set_pixel_color_hsv(i, h + i * 30, 255, 200);
            }
            ws281x_ir_show_leds();
            vTaskDelay(pdMS_TO_TICKS(30));
        }

        // 呼吸效果
        for (float b = 0.0; b <= 1.0; b += 0.1) {
            ws2812_set_global_brightness(b);
            ws281x_ir_show_leds();
            vTaskDelay(pdMS_TO_TICKS(50));
        }
        for (float b = 1.0; b >= 0.0; b -= 0.1) {
            ws2812_set_global_brightness(b);
            ws281x_ir_show_leds();
            vTaskDelay(pdMS_TO_TICKS(50));
        }
    }
#endif
}
```

### 逐颗点亮（跑马灯）

```c
void chase_animation(void)
{
    ws2812_init(&g_ws2812_strip);
    ws2812_set_all_pixels_color(0, 0, 0, 1.0);

    while (1) {
        for (uint8_t i = 0; i < ws2812_get_led_count(); i++) {
            ws2812_set_pixel_color(i, 255, 255, 255);
            ws2812_show_leds();
            vTaskDelay(pdMS_TO_TICKS(100));
            ws2812_set_pixel_color(i, 0, 0, 0);  // 熄灭
        }
    }
}
```
