# wl80211 Wi-Fi 驱动接口

## 概述

`wl80211` 是 BL616/BL618 Wi-Fi 驱动核心模块，位于 `components/wireless/wl80211/`，提供 IEEE 802.11 无线局域网的完整驱动支持。它是 `wifi_mgmr` 上层 API 的底层支撑，直接操作 MAC 软件层（macsw）和 WPA Supplicant。

## 架构层次

```
+---------------------------+
|     Application / wifi_mgmr API  |
+---------------------------+

       +------------+
```

## 头文件

```c
#include "wl80211.h"
```

## 全局状态

```c
extern struct wl80211_global_state wl80211_glb;
```

`wl80211_glb` 包含驱动全局状态：

| 字段 | 类型 | 说明 |
|------|------|------|
| `country` | `const struct ieee80211_dot_d *` | 国家码配置 |
| `scanning` | bit | 正在扫描 |
| `connecting` | bit | 正在连接 |
| `associated` | bit | 已关联 |
| `authenticating` | bit | 正在认证 |
| `link_up` | bit | 连接建立 |
| `ap_en` | bit | AP 模式使能 |
| `monitor_en` | bit | 监听模式使能 |
| `connect_params` | `struct wl80211_connect_params *` | 当前连接参数 |
| `aid` | `uint16_t` | Association ID |

## 初始化

```c
void wl80211_init(void);
```

初始化 wl80211 驱动。建议通过 `wifi_mgmr_init()` 间接调用，而非直接调用。

## 事件类型

```c
enum {
    WL80211_EVT_SCAN_DONE,        // 扫描完成
    WL80211_EVT_STA_CONNECTED,    // STA 连接成功
    WL80211_EVT_STA_DISCONNECTED, // STA 断开
    WL80211_EVT_AP_STARTED,       // AP 已启动
    WL80211_EVT_AP_STOPPED,       // AP 已停止
    WL80211_EVT_AP_STA_ADDED,     // AP 下有 STA 加入
    WL80211_EVT_AP_STA_DEL,       // AP 下有 STA 离开
};
```

事件通过 `wl80211_post_event()` 异步投递：

```c
void wl80211_post_event(int code1, int code2);
```

## 控制命令

`wl80211_cntrl()` 是驱动控制的中央接口：

```c
int wl80211_cntrl(int cmd, ...);
```

命令列表：

| 命令 | 说明 | 参数 |
|------|------|------|
| `WL80211_CTRL_SCAN` | 启动扫描 | `struct wl80211_scan_params *` |
| `WL80211_CTRL_STA_CONNECT` | STA 连接 | `struct wl80211_connect_params *` |
| `WL80211_CTRL_STA_DISCONNECT` | STA 断开 | 无 |
| `WL80211_CTRL_STA_SET_PS_MODE` | 设置功耗模式 | `int mode` |
| `WL80211_CTRL_STA_GET_MAC` | 获取 STA MAC | `uint8_t mac[6]` |
| `WL80211_CTRL_STA_GET_RSSI` | 获取 RSSI | `int *rssi` |
| `WL80211_CTRL_AP_START` | 启动 AP | `struct wl80211_ap_settings *` |
| `WL80211_CTRL_AP_STOP` | 停止 AP | 无 |
| `WL80211_CTRL_AP_SET_SSID` | 设置 AP SSID | `uint8_t ssid[33]` |
| `WL80211_CTRL_AP_SET_PASSWORD` | 设置 AP 密码 | `uint8_t password[65]` |
| `WL80211_CTRL_MONITOR_START` | 启动监听模式 | `struct wl80211_monitor_settings *` |
| `WL80211_CTRL_MONITOR_STOP` | 停止监听模式 | 无 |
| `WL80211_CTRL_INJECT_FRAME` | 注入 802.11 帧 | buffer, len |

## 便捷内联函数

### 扫描

```c
static inline int wl80211_scan(struct wl80211_scan_params *param);
```

返回：`0` 成功，`-1` 国家码未设置，`-2` 监听模式下不允许扫描。

### STA 连接

```c
static inline int wl80211_sta_connect(struct wl80211_connect_params *param);
```

返回：`0` 成功，`-1` 监听模式下不允许连接，`-2` 连接已在进行中，`-3` 无效 SSID，`-4` 无效信道，`-5` 无效 IE 参数。

### STA 断开

```c
static inline int wl80211_sta_disconnect(void);
```

### STA 连接状态

```c
static inline int wl80211_sta_is_connected(void);  // 返回 1=已连接，0=未连接
static inline int wl80211_sta_get_bssid(uint8_t bssid[6]);
static inline int wl80211_sta_get_channel(void);
```

### 功耗管理

```c
static inline int wl80211_sta_set_ps(int mode);
```

### 信道/频率转换

```c
static inline int wl80211_channel_to_freq(int channel);
static inline int wl80211_freq_to_channel(int freq);
static inline int wl80211_freq_check(int freq);  // 返回 -1=无效，0=2.4G，1=5G
static inline int wl80211_channel_check(int channel);
```

## 工作代码示例

### STA 扫描与连接

```c
#include "wl80211.h"
#include "wifi_mgmr_ext.h"
#include <string.h>

static void wifi_event_cb(int code, int val, void *arg)
{
    switch (code) {
    case CODE_WIFI_ON_CONNECTED:
        printf("Wi-Fi connected\r\n");
        break;
    case CODE_WIFI_ON_DISCONNECT:
        printf("Wi-Fi disconnected\r\n");
        break;
    case CODE_WIFI_ON_SCAN_DONE:
        printf("Scan done, found %d APs\r\n", val);
        break;
    case CODE_WIFI_ON_GOT_IP:
        printf("Got IP\r\n");
        break;
    }
}

void wifi_sta_example(void)
{
    struct wl80211_scan_params scan_params = { 0 };
    struct wl80211_connect_params conn_params = { 0 };

    /* 设置国家码（扫描前必须） */
    wl80211_cntrl(WL80211_CTRL_SET_COUNTRY, "CN");

    /* 启动被动扫描 */
    scan_params.flags = WL80211_SCAN_FLAGS_PASSIVE;
    scan_params.probe_cnt = 3;
    scan_params.probe_interval = 100; /* TU */
    int ret = wl80211_scan(&scan_params);
    printf("scan ret=%d\r\n", ret);

    /* 连接 AP */
    memset(&conn_params, 0, sizeof(conn_params));
    strncpy((char *)conn_params.ssid, "MyAP", 32);
    strncpy((char *)conn_params.password, "password123", 64);
    conn_params.auth_type = WL80211_AUTHTYPE_OPEN_SYSTEM;
    conn_params.mfp = WL80211_MFP_NO;
    conn_params.channel = 6;

    ret = wl80211_sta_connect(&conn_params);
    printf("connect ret=%d\r\n", ret);

    /* 检查连接状态 */
    if (wl80211_sta_is_connected()) {
        uint8_t bssid[6];
        int channel = wl80211_sta_get_channel();
        wl80211_sta_get_bssid(bssid);
        printf("Connected to %s, channel=%d\r\n", conn_params.ssid, channel);
    }
}
```

### 监听模式

```c
static void monitor_rx_cb(void *ctx, void *pkt, size_t len,
                          size_t mac_hdr_len, int rssi)
{
    printf("RX frame: len=%u, rssi=%d\r\n", (unsigned)len, rssi);
    /* 处理原始 802.11 帧 */
}

void wifi_monitor_example(void)
{
    struct wl80211_monitor_settings mon = {
        .channel_width = WL80211_CHAN_WIDTH_20,
        .center_freq1 = 2437, /* channel 6 */
        .recv_ctx = NULL,
        .recv = monitor_rx_cb,
    };

    wl80211_cntrl(WL80211_CTRL_MONITOR_START, &mon);
    printf("Monitor mode started on channel 6\r\n");

    /* ... 接收处理 ... */

    wl80211_cntrl(WL80211_CTRL_MONITOR_STOP);
}
```

## 注意事项

1. **国家码必须在扫描前设置**，否则 `wl80211_scan()` 返回 `-1`
2. **监听模式**和 **STA/AP 模式互斥**，切换前需先停止当前模式
3. `wl80211_glb.link_up` 仅在成功关联（收到 Assoc Resp）后置位
4. WPA3（SAE）连接需设置 `auth_type = WL80211_AUTHTYPE_SAE`
5. 建议使用 `wifi_mgmr_*` API 而非直接操作 `wl80211_cntrl()`，除非需要高级控制
